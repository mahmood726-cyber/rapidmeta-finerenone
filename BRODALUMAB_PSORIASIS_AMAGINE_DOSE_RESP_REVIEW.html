Outcome selection (PASI 75 over sPGA 0/1): PASI 75 and sPGA 0/1 are both registered AACT co-primaries in all 3 trials. PASI 75 was chosen for dose-response slope estimation because (a) it is the standard dose-response endpoint in psoriasis literature, (b) PASI 75 is a more granular 75 %-improvement threshold than sPGA's 5-category clear/almost-clear collapse, (c) PASI 75 events are larger numbers, giving better small-sample stability of the GL covariance. Both endpoints would yield the same monotonic signal — per-trial PASI 75 event-rates rise strictly with dose in all 3 trials (AMAGINE-1: 2.7 % → 60.3 % → 83.3 %; AMAGINE-2: 8.1 % → 66.6 % → 86.3 %; AMAGINE-3: 6.0 % → 69.2 % → 85.1 %).

Per-trial RCS feasibility (drives the engine-vs-R disagreement): The 3-knot Harrell-default RCS needs ≥ 3 distinct non-zero dose levels across the pool for the percentile knots to land on real data. This fixture has only 2 distinct non-zero doses ({140, 210} mg) and 3 distinct doses in total ({0, 140, 210} mg). The engine's fitRCS calls rcsKnots on the unique positive doses, finds fewer than K = 3 distinct knot locations, and short-circuits to fitLinear with fallback='degenerate_to_linear' and rcs=null. R dosresmeta proceeds anyway and places its 3 percentile knots inside the 140–210 mg range (knots = [157.5, 175, 192.5]) — no interior data, but a fit is returned. Methods (click to expand)

This review pools 3 Phase 3 RCTs of brodalumab in adults with moderate-to-severe plaque psoriasis, all from the Amgen AMAGINE programme (2015–2016). All trials randomized to brodalumab 140 mg or 210 mg subcutaneous Q2W with a loading dose at Week 1, measured PASI 75 response at Week 12, and were registered prospectively on ClinicalTrials.gov.

Within-drug dose-response: All 3 trials use the same molecule (brodalumab), same route (subcutaneous Q2W with loading dose at Week 1), same duration (12-week induction phase), and same population (adults with moderate-to-severe plaque psoriasis, BSA ≥ 10 %, sPGA ≥ 3, PASI ≥ 12, eligible for systemic therapy or phototherapy). The pooled slope is therefore a within-drug PASI 75 dose-response curve with no cross-drug equipotency assumption. The 140 mg and 210 mg dose levels are the two Phase 3 induction doses; the 210 mg arm is the FDA-approved psoriasis dose. AMAGINE-2 and AMAGINE-3 differ from AMAGINE-1 only in having an additional ustekinumab active-comparator arm (dropped per Option A), so the kept per-trial structure is identical 3-arm placebo + 140 + 210 mg across all 3 trials.

Binary-outcome path (engine v0.4): The engine's fitLinear takes per-arm events and n directly and constructs the Greenland-Longnecker shared-reference covariance for log-RR contrasts (off-diagonal = 1/eventref − 1/nref; diagonal = 1/eventi − 1/ni + 1/eventref − 1/nref). No SE→SD conversion is needed (unlike the continuous-outcome flagships SGLT2I / TIRZEPATIDE / SUSTAIN / SELECT). The pooled slope is log-RR per mg, back-transformed to RR per mg for KPI display. F-1 conditional zero-cell continuity correction is wired in the engine but does not fire on this fixture (every trial has > 0 events in every arm: AMAGINE-1 placebo has 6 events out of 220; the smallest event count is well above zero).

Active-comparator handling (Option A — drop, do not down-code as dose=0): AMAGINE-2 and AMAGINE-3 randomized brodalumab vs placebo vs ustekinumab. The ustekinumab arms are DROPPED from the dose-response arms list (210/300 events in AMAGINE-2 OG003 = 70 %, 217/313 in AMAGINE-3 OG003 = 69 %; preserved as _dropped_comparator_arm audit metadata in the fixture). Placebo is the dose=0 reference in all 3 trials. AMAGINE-1 had no active comparator, so its 3-arm structure is unchanged.

Engine RCS refusal (the methodological story, expanded in Tab 2): The 3-knot Harrell-default RCS positions knots at the 10th, 50th, and 90th percentiles of the unique positive doses (per rcsKnots(allDoses, 3) with the default Harrell percentile placement). With only 2 distinct positive doses (140 and 210 mg), the engine's rcsKnots returns fewer than K = 3 distinct knot locations, and fitRCS short-circuits with layer='linear', fallback='degenerate_to_linear', rcs=null. R dosresmeta does NOT short-circuit on this case — it proceeds to compute percentile knots from the design matrix and returns knots = [157.5, 175, 192.5] (the 10th, 50th, 90th percentiles of {140, 210}-based design points), spline_coefs = [0.01763, -0.01875], and a highly significant nonlinearity_wald_p = 1.64e-13. The R fit is technically valid arithmetic but interpretively dubious: there are no data points between 140 and 210 mg to support the within-range curvature the spline encodes. The engine's refusal is the methodologically conservative call.

Why both engines refused on prior flagships but disagree here: The SUSTAIN flagship (Round 3.7, 6 trials) had R refuse RCS via the sparse-per-trial-arm requirement (4 of 6 trials had only 1 non-reference arm). The SELECT flagship (Round 3.8, 4 trials) had R refuse RCS via the same sparse-per-trial-arm error. On AMAGINE all 3 trials have 2 non-reference arms (140 and 210 mg), so R's sparse-arm guard does not fire — but the cross-trial dose grid still has only 2 distinct positive doses. The engine checks the cross-trial knot count (the constraint the spline actually needs); R checks the per-trial arm count (a related but distinct constraint that does not catch this case). Engine v0.4 is therefore more conservative than R dosresmeta 2.2.0 on this specific dose-grid topology.

Estimand uniformity: All 3 trials use the FDA-pre-registered primary estimand (treatment-policy with non-responder imputation for missing data). Per-arm denominators reflect the modified intention-to-treat set with NRI per protocol. AACT registers COUNT_OF_PARTICIPANTS with units='Participants' for all 3 PASI 75 outcomes — direct event-count extraction is appropriate, with denominators from outcome_counts.txt Measure-scope rows.

Statistical methods match the engine v0.4 conventions: two-stage Greenland-Longnecker linear pool with REML + HKSJ (HKSJ floor max(1, Q/(k-1))); Q-profile τ2 CI (Viechtbauer 2007); PI uses tk-1 df (Cochrane Handbook v6.5). The RCS layer is requested but the engine returns the documented fallback (layer='linear', fallback='degenerate_to_linear', no .rcs block) because the 2-distinct-positive-dose grid does not support a 3-knot Harrell basis. R dosresmeta linear layer matches the engine within |Δ| < threshold (linear-slope row GREEN). R dosresmeta RCS fits but uses degenerate knots; the R-parity badge (Tab 4) renders those 3 RCS rows in a custom “engine-declined / R-fit (uninformative knots)” panel rather than as GREEN / AMBER / DEFERRED. R lme4::glmer one-stage Poisson-with-offset on the per-arm event counts is reported as a sensitivity (Tab 3) — R reports the fit as converged=false (random-effects variance pinned at the boundary), reflecting that between-trial variance of the dose coefficient is too small to identify a positive random-effects variance on a strictly-monotonic 3-trial pool.

Future-extension note (AMAGINE-1 Phase 2b): An earlier Phase 2b dose-finding study of brodalumab in psoriasis randomized to 70 / 140 / 210 / 280 mg + placebo (n ~ 198), which would provide 4 distinct positive doses and unblock the 3-knot RCS. That study is not part of the AMAGINE Phase 3 programme indexed in AACT under these three NCTs and is excluded from this fixture. A future review pooling AMAGINE Phase 3 + Phase 2b would let the engine fit a real RCS; on the current 3-trial Phase 3 pool the engine's refusal is the right call.
