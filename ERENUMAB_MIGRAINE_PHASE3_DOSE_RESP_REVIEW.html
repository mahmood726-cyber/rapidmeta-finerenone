Methodology demonstration flagship (Round 3.10 mirrors Round 3.8 SELECT — both engine and R refuse RCS): This is a 3-trial within-drug erenumab Δ MMD dose-response pool on the dose grid {0, 70, 140} mg with sparse per-trial coverage. STRIVE alone tests both 70 mg and 140 mg vs placebo; ARISE tests only 70 mg vs placebo; LIBERTY tests only 140 mg vs placebo. The pool therefore has only 2 distinct positive doses ({70, 140} mg) and only 1 of the 3 trials (STRIVE) contributes both doses. The 3-knot Harrell-default restricted cubic spline basis needs ≥ K = 3 distinct knot locations across the unique positive doses for the design matrix to be non-singular under two-stage pooling; only 2 distinct positive doses exist. The engine returns layer='linear', fallback='degenerate_to_linear', with no .rcs block — this is the engine's documented safety net for coarse dose grids, not a bug. R dosresmeta refuses RCS on the same fixture with the parallel error (“each study provides at least p non-referent obs”) because ARISE and LIBERTY each have only 1 non-reference arm versus Kp = 2 spline coefficients required. Engine and R agree — the dose grid does not support a 3-knot RCS in two-stage form. The linear pool succeeds at k=3 and matches R within |Δ| < threshold. Tab 2 surfaces the degeneration explicitly; Tab 3's R-parity badge therefore reduces to the linear-pool rows only.
Population heterogeneity caveat (treatment-refractory vs treatment-naive): STRIVE and ARISE enrolled treatment-naive adults with episodic migraine (4–14 MMD; no prior failure of multiple classes of preventives). LIBERTY enrolled a treatment-refractory population — adults with episodic migraine AND prior failure of 2 to 4 oral preventive treatments (39% had failed 2 preventives, 38% had failed 3, 23% had failed 4). This produces a striking baseline-population effect: LIBERTY's placebo arm reduces Δ MMD by only −0.16 days vs −1.83 / −1.84 days in STRIVE / ARISE. The dose-equivalence assumption (same molecule, same SC route, same once-monthly schedule) is NOT violated — the dose-response axis remains valid — but the placebo-arm response is a function of the prior-failure population and is the dominant source of LIBERTY's smaller absolute treatment effect on this endpoint. Per-trial monotonicity is preserved in all 3 trials (STRIVE pbo → 70 → 140 = −1.83 → −3.23 → −3.67; ARISE pbo → 70 = −1.84 → −2.88; LIBERTY pbo → 140 = −0.16 → −1.75). Each individual trial is internally consistent on the dose-response axis.

Methods (click to expand)

This review pools 3 Phase 3 RCTs of erenumab (AMG334, a fully human IgG2 monoclonal antibody against the calcitonin gene-related peptide receptor) in adults with episodic migraine. All 3 trials randomised participants to once-monthly subcutaneous erenumab vs placebo and measured Δ monthly migraine days (MMD) change-from-baseline at the end of the double-blind treatment phase.

Within-drug dose-response: All 3 trials use the same molecule (erenumab), same route (subcutaneous once-monthly injection), and the same FDA-evaluated dose range (70 mg and 140 mg). The pooled slope is therefore a within-drug Δ MMD dose-response curve and has no cross-drug equipotency assumption. STRIVE contributes the only within-trial 70-vs-140 mg contrast; ARISE pins the placebo ↔ 70 mg leg in a separate trial, and LIBERTY pins the placebo ↔ 140 mg leg in a separate trial. The cross-trial 70-vs-140 mg comparison therefore depends on assuming dose-equivalence across the 3 trials (same molecule, same route, comparable double-blind windows of 12–24 weeks).

Active-comparator handling: Not applicable. All 3 trials are placebo-controlled with no active-comparator arms; placebo is the dose=0 reference in all 3 trials. Unlike SUSTAIN / SURPASS / SELECT / AMAGINE fixtures, no Option A active-comparator-drop is required here.

Engine RCS-fallback (the headline methodological story): The 3-knot Harrell-default RCS positions knots at the 10th, 50th, and 90th percentiles of the unique positive doses. With only 2 distinct positive doses in the pool ({70, 140} mg), rcsKnots(allDoses, 3) returns fewer than the K = 3 required distinct knot locations and fitRCS short-circuits to fitLinear, decorating the result with layer='linear', fallback='degenerate_to_linear', and rcs=null. R dosresmeta refuses to fit RCS on the same fixture with the parallel error “A two-stage approach requires that each study provides at least p non-referent obs” — ARISE has only 1 non-reference arm (70 mg) and LIBERTY has only 1 non-reference arm (140 mg) versus Kp = 2 spline coefficients required. Engine and R agree the dose grid does not support a 3-knot RCS in two-stage form. The linear two-stage layer (k=3) is unaffected and is the methodologically primary result for this fixture.

Estimand uniformity: MMRM (mixed-effects model for repeated measures) with multiple imputation per protocol-specified statistical analysis plan, treatment-policy primary estimand, used consistently across all 3 trials. STRIVE and ARISE register the continuous Δ MMD as param_type=LEAST_SQUARES_MEAN (LSM) reflecting the MMRM analysis. LIBERTY registers the continuous secondary Δ MMD as param_type=MEAN (the registered primary on the binary ≥50% responder rate uses logistic regression; the secondary Δ MMD MEAN is the MMRM-derived estimate labelled MEAN in AACT). AACT registers a single estimand row per result_group per outcome_id in all 3 trials (no on-treatment vs in-trial split).

Dispersion-convention note: All 3 trials report ± Standard Error in AACT (STRIVE and ARISE as LSM ± SE; LIBERTY as MEAN ± SE). The fixture computes sd = se × √n using the per-arm analysed N from outcome_counts.txt Measure-scope rows. Raw SE preserved per arm as _aact_se for audit. The engine's mdCovariance() uses sd2/n which equals SE2 — invariant to MEAN-vs-LSM labelling at the variance level.

Population heterogeneity (treatment-refractory placebo response): LIBERTY's placebo arm reduces Δ MMD by −0.16 days (vs −1.83 / −1.84 in STRIVE / ARISE) because the LIBERTY population had failed 2–4 prior oral preventives — a known feature of treatment-refractory cohorts, not an extraction artefact. The dose-response axis is unaffected: each trial is internally monotonic and the pooled slope is identified from the within-trial dose-vs-placebo contrasts. The cross-trial heterogeneity from the placebo-response shift is absorbed into the per-trial intercept (the dose-vs-placebo contrast) by construction in the GL two-stage formulation.

Statistical methods match the engine v0.4 conventions: two-stage Greenland-Longnecker linear pool with REML+HKSJ (HKSJ floor max(1, Q/(k-1))); Q-profile τ2 CI (Viechtbauer 2007); PI uses tk-1 df (Cochrane Handbook v6.5). The RCS layer is requested but the engine returns the documented fallback (layer='linear', fallback='degenerate_to_linear', no .rcs block) because the 2-distinct-positive-dose grid does not support a 3-knot Harrell basis under two-stage pooling. R dosresmeta linear layer matches the engine within |Δ| < threshold; R RCS refused with a parallel sparse-arm error; the R-parity badge therefore reduces to the linear-pool rows only.
