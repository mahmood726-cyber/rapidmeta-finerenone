## Methodology

- API: `https://clinicaltrials.gov/api/v2/studies/{NCT}?format=json` (no auth, polite 0.15s sleep between calls)
- Cache: `outputs/ctgov_cache.json` so re-runs are idempotent
- Outlier threshold: ratio outside 0.30–1.10 (allows for 2-of-3-arm contrasts, mITT vs randomized differences, and per-protocol subset analyses)
- Swap detection: within each file, for each pair (A, B), checks if `|claim_A - ctg_B|` is meaningfully smaller than `|claim_A - ctg_A|` AND vice versa. Threshold: 0.15 absolute improvement on both sides.
- Limitations: doesn't catch swaps in 3+ trial families (e.g., 3 trials cycled through wrong NCTs); doesn't verify trial design / drug / population.
