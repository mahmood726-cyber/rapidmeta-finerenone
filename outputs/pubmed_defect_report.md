## How to fix each

- Wrong PMID, correct NCT: search PubMed for trial acronym + drug, replace `pmid:` in realData
- Wrong NCT, correct PMID: PubMed databank record gives correct NCT, replace key + AUTO_INCLUDE_TRIAL_IDS + nctAcronyms
- Both wrong: re-research from published paper

After each fix, re-run both `aact_cross_check_v2.py` and `pubmed_nct_linkage_check.py`.
Both must be 0-defect for the affected file before push.
