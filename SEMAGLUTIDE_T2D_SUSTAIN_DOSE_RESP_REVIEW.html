Per-trial RCS feasibility (drives the methodology divergence): R dosresmeta requires ≥ Kp + 1 = 3 arms per trial for the 3-knot RCS (2 non-reference arms minimum). Only SUSTAIN-1 and SUSTAIN-5 satisfy this (3 arms each: placebo + 0.5 + 1.0 mg). Methods (click to expand)

This review pools 6 Phase 3 RCTs of semaglutide in adults with type 2 diabetes (T2D), all from the Novo Nordisk SUSTAIN programme (2017-2021). All trials randomized to once-weekly subcutaneous semaglutide vs a control (placebo or active comparator), measured HbA1c change-from-baseline at 30, 40, or 56 weeks, and were registered prospectively on ClinicalTrials.gov.

Within-drug dose-response: All 6 trials use the same molecule (semaglutide) and same route (subcutaneous once-weekly). The pooled slope is therefore a within-drug dose-response curve and has no cross-drug equipotency assumption. The 0.5, 1.0, and 2.0 mg dose levels span the FDA-approved semaglutide T2D dosing range. SUSTAIN-1 + SUSTAIN-5 contribute the placebo → 0.5 mg and 0.5 → 1.0 mg transitions; SUSTAIN-FORTE alone contributes the 1.0 → 2.0 mg transition.

Active-comparator handling (Option A — drop, do not down-code as dose=0): SUSTAIN-2 / SUSTAIN-4 / SUSTAIN-7 randomized semaglutide vs active comparators (sitagliptin 100 mg, insulin glargine titrated, dulaglutide 0.75 + 1.5 mg respectively). Active comparator arms are DROPPED from the dose-response arms list — the engine treats the lowest semaglutide dose (0.5 mg) as the per-trial reference (is_reference=true) for those 3 trials. SUSTAIN-1 + SUSTAIN-5 were placebo-controlled, so placebo is the reference (dose=0). SUSTAIN-FORTE is a head-to-head 2.0 vs 1.0 mg with NO placebo arm; semaglutide 1.0 mg is the per-trial reference (active-control kept because it IS one of the doses on the dose-response curve, just at the lowest level for that trial). The comparator metadata (comparator, comparator_label, comparator_lsm_change_pct) is preserved in the trial-level fixture as the audit trail.

Engine vs R RCS-feasibility divergence (the headline methodological story): R dosresmeta refuses to fit the RCS layer with the error “A two-stage approach requires that each study provides at least p non-referent obs”. R requires Kp + 1 = 3 arms minimum per trial for the 3-knot RCS (2 non-reference arms). After Option A dropping, only SUSTAIN-1 + SUSTAIN-5 satisfy this. The engine's fitRCS drops singular per-trial trials transparently inside its inner try/catch on matInv(XtSX) and pools the surviving kRCS=2 via full multivariate REML on the Cholesky parameters of the Kp × Kp τ2 matrix. This is methodologically more permissive than R but the per-trial information is identical — the engine simply doesn't error when a per-trial RCS design is singular. The linear two-stage layer (klinear=6) is unaffected and matches R within |Δ| < 0.01.

Estimand uniformity (treatment-policy as primary, trial-product for SUSTAIN-FORTE): SUSTAIN-1/2/4/5/7 use the AACT-registered MMRM treatment-policy estimand. For SUSTAIN-FORTE, AACT registers both on-treatment-without-rescue-medication (trial-product estimand) and in-trial (treatment-policy estimand) for the same outcome_id. The fixture uses the trial-product row (-2.0 / -2.2 percentage points for 1 / 2 mg) because the abstract headline and FDA labelling pivot on it; the in-trial estimand (-1.9 / -2.2) differs by ≤ 0.1 percentage points and would not change the qualitative dose-response shape.

Dispersion-convention note: AACT mixes two dispersion conventions across the 6 SUSTAIN trials: SUSTAIN-1 and SUSTAIN-FORTE register MEAN ± Standard Deviation (raw observed); SUSTAIN-2/4/5/7 register LEAST_SQUARES_MEAN ± Standard Error (MMRM). The fixture converts SE to SD via sd = se × √n per the AACT per-arm N. The engine's mdCovariance() uses sd2/n, which is invariant to the re-expression — both branches produce identical contrast variances. Documented in fixture dispersion_handling.

Statistical methods match the engine v0.3.0 conventions: two-stage Greenland-Longnecker linear pool with REML+HKSJ (HKSJ floor max(1, Q/(k-1))); restricted cubic splines with 3 Harrell-default knots (placed at the 10th, 50th, 90th percentiles of the unique non-zero doses); full multivariate REML for the RCS τ2 matrix (Nelder-Mead on Cholesky parameters); HKSJ-multivariate + tk-1 CIs on RCS pooled output; Q-profile τ2 CI on fitLinear (Viechtbauer 2007); one-stage hierarchical (HbA1c, continuous, lme4::lmer with bobyqa optimizer) computed by R and surfaced from precomputed JSON. PI uses tk-1 df (Cochrane Handbook v6.5). At the RCS layer, the engine effectively reduces to kRCS=2 because 4 of 6 trials have singular per-trial RCS design matrices (1 non-reference arm vs Kp=2 spline coefficients); tcrit = qt(0.975, 1) = 12.706 reflects this reduced k.

Continuous-outcome support: Engine v0.2.0+ continuous branch in fitLinear/fitRCS uses a per-trial covariance matrix mirroring the binary GL shared-reference structure (off-diagonal = sdref2/nref; diagonal = sdi2/ni + sdref2/nref). Per-arm SDs derived from AACT-reported SE via sd = se × √n where applicable. Linear-layer parity validated against R dosresmeta(type="md") in the R-parity badge tab.
